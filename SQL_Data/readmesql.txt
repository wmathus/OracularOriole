T2D_Genetics Database Schema & Cloud Server Access

This folder contains the T2D_Genetics MySQL database schema, designed to store and analyze genetic data related to Type 2 Diabetes (T2D). The database is hosted on a cloud MySQL
server to allow remote access.

1.Prerequisites
Before running the script, make sure you have:
MySQL Workbench installed -> https://dev.mysql.com/downloads/workbench/

2.Connecting to the Cloud Database
To connect to the cloud MySQL database using MySQL Workbench:

Open MySQL Workbench.
Click Database > Manage Connections > New.
Fill in the details:

| Parameter         | Value                        |
|------------------|----------------------------|
| **Host**        | sql8.freesqldatabase.com    |
| **Database Name** | sql8762368                  |
| **Username**     | sql8762368                  |
| **Password**     | DB45Pl1pRv                  |
| **Port Number**  | 3306                         |

Click "Test Connection" to verify access.
Click "OK" and select the saved connection to open the database.
